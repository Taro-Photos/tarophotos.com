# 23 SEO & 構造化データ（抜粋）

- 画像付近にキャプション/解説を配置、`alt` は文脈に即して記述。
- `Person/Organization`（/about）に `sameAs` で SNS を紐付け。
- 各作品に `ImageObject`（`contentUrl/creator/contentLocation/datePublished`）。
- Services に FAQ（`FAQPage`）を導入。
- OG 画像は 1200×630px 推奨、X は `summary_large_image`。
- UX と SEO の両立：無限スクロールでも**内部ページネーション URL**を用意。
