# 10 情報設計 / ルーティング

- `/` Home（世界観の凝縮 + 代表シリーズ導線）
- `/works` Works Index（Series 索引）
- `/works/[series]` Series 詳細（写真/解説/Exif/Behind）
- `/services` 受注撮影（メニュー/料金/FAQ/実績）
- `/booking` 相談・見積（フォーム + 日程調整）
- `/about` 作家情報（受賞/掲載/機材/クライアント）
- `/journal` 記事（制作記・現場レポ）
- `/clients` 納品（2025-09 公開停止。必要時は社内共有リンクを個別発行）
